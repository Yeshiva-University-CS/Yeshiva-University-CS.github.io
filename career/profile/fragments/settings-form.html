GitHub Settings

Connect your GitHub account and link your YU repository to load and save your profile.

Token is stored only in your browser's local storage — never sent to any server other than GitHub.
Required permission: repo scope.

Format: FirstName_LastName_800XXXXXX (e.g., John_Doe_800123456)

How to set up:

  1. 1 Go to GitHub Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. 2 Click "Generate new token (classic)", set expiration to at least 90 days, enable the repo scope
  3. 3 Your repo name format: FirstName_LastName_800XXXXXX — find it on your GitHub profile under Yeshiva-University-CS
  4. 4 Paste your token and repo name above, then click "Save Settings & Load Profile"
