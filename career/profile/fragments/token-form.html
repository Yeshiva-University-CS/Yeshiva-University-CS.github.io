Connect Your GitHub Account

To load and save your profile, you need a GitHub Personal Access Token with repo scope. Your token is stored only in this browser — it is never sent to any server other than GitHub.

Create a new token with repo scope.

How to create a token:

  1. Click the link above to open GitHub token settings
  2. Set the expiration to at least 90 days
  3. Make sure repo is checked under Select scopes
  4. Click Generate token and copy it here
